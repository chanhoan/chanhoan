- Kafka, Redis, Docker 같은 기술을 직접 적용해보며 배우는 걸 즐깁니다.
- 코드는 결국 문제를 푸는 도구라고 생각하며, 복잡한 현실을 더 단순하게 만들고 싶습니다. 🔥
